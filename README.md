# Dmj Task
Hello! This is a simple brief on this app, which is a task management app with firebase
It contains a set of features.

 - Splash Screen
 - Login Screen 
 - Sign Up Screen 
 - Home Screen For get all Task For You and Other User 
 - Add Task Screen 
 - Edit Task Screen 
 - Notification Feature 
 - Logout
